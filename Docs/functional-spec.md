# 機能仕様

## 画面一覧

| 画面 | パス案 | 説明 |
| --- | --- | --- |
| 本一覧 | `/books` | 登録済みの本を一覧表示し、検索と絞り込みを行う |
| 本登録 | `/books/new` | 新しい本を登録する |
| 本詳細 | `/books/:id` | 1冊の詳細情報を表示する |
| 本編集 | `/books/:id/edit` | 登録済みの本を編集する |

MVPではトップページ `/` は `/books` に誘導します。

## 本一覧

### 表示項目

- タイトル
- 著者
- 読書ステータス
- 所有ステータス
- ISBN
- 保管場所
- 更新日時

### 操作

- キーワード検索
- 読書ステータスによる絞り込み
- 所有ステータスによる絞り込み
- 詳細画面への遷移
- 登録画面への遷移

### 検索対象

- タイトル
- 著者
- ISBN

## 本登録

### 入力項目

| 項目 | 必須 | 説明 |
| --- | --- | --- |
| タイトル | 必須 | 本の名前 |
| 著者 | 任意 | 複数名はカンマ区切りで入力 |
| 出版社 | 任意 | 出版社名 |
| 出版日 | 任意 | `YYYY-MM-DD` 形式 |
| ISBN | 任意 | ISBN-10またはISBN-13 |
| 読書ステータス | 必須 | 初期値は `unread` |
| 所有ステータス | 必須 | 初期値は `owned` |
| 保管場所 | 任意 | 本棚、部屋、職場など |
| メモ | 任意 | 自由記述 |

### バリデーション

- タイトルは1文字以上、200文字以下
- ISBNは未入力、ISBN-10、ISBN-13のいずれかを許可
- 出版日は未入力、または有効な日付のみ許可
- メモは5000文字以下

## 本詳細

### 表示項目

- タイトル
- 著者
- 出版社
- 出版日
- ISBN
- 読書ステータス
- 所有ステータス
- 保管場所
- メモ
- 登録日時
- 更新日時

### 操作

- 編集画面への遷移
- 一覧画面へ戻る
- 削除

## 本編集

登録時と同じ項目を編集できます。

### 保存時の振る舞い

- 入力値が有効な場合は保存し、詳細画面へ戻ります。
- 入力値が不正な場合は保存せず、エラーメッセージを表示します。

## 本削除

- 削除前に確認ダイアログを表示します。
- MVPでは物理削除とします。
- 将来的に削除履歴が必要になった場合は論理削除へ変更します。

## ステータス

### 読書ステータス

| 値 | 表示名 | 説明 |
| --- | --- | --- |
| `unread` | 未読 | まだ読んでいない |
| `reading` | 読書中 | 現在読んでいる |
| `finished` | 読了 | 読み終わった |
| `stacked` | 積読 | 購入済みで後で読む予定 |
| `reread` | 再読したい | もう一度読みたい |

### 所有ステータス

| 値 | 表示名 | 説明 |
| --- | --- | --- |
| `owned` | 所有中 | 現在所有している |
| `wishlist` | 欲しい | まだ所有していないが欲しい |
| `released` | 手放した | 売却、譲渡、処分済み |
